# Consolidated Repository Snapshot

This directory contains a repository-wide consolidation generated by the consolidation script.

## Files generated
- consolidated_repository_files.txt
- consolidated_repository_files_index.txt
- repository_directory_tree.txt

## Exclusions
The snapshot intentionally excludes:
- .venv
- .git
- docs
- logs
- scripts
- .gitignore
- .env
- LICENSE / LICENCE
- README.md / readme.md

## Usage
Run with:
```bash
uv run python scripts/consolidate_all.py --output-dir ./consolidated/
```
